# Testing UI Web Sauce Demo

Web UI testing using Cucumber, Java, Selenium and https://www.saucedemo.com/ as the website

## FEATURE
* LOGIN
* PRODUCT
* SORTING
* MENU
* CART
* CHECKOUT

## FRAMEWORK
* TestNG

## LIBRARY
* Cucumber
* Selenium

## How to Start and Run This Automation Project
